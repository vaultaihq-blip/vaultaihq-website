Política de Privacidade — Pick&Pay

Última atualização: maio de 2026

Desenvolvido por Vault AI HQ · vault.ai.hq@gmail.com

1. Dados que recolhemos

  • Nome e email (para criar conta)
  • Câmara e fotografias (para fazer scan de recibos via OCR)
  • Dados de recibos (itens e valores, para divisão de contas)
  • Tokens de notificações push (para convites de sessão)
  • Histórico de compras de créditos (via Google Play / App Store)

2. Para que usamos os dados

  • Fornecer o serviço de divisão de contas
  • Processar compras in-app via RevenueCat
  • Enviar notificações de sessão
  • Melhorar a app e prevenir abusos

3. Terceiros

As imagens de recibos são processadas por:

  • OpenAI — análise OCR — openai.com/privacy
  • Supabase — base de dados e autenticação — supabase.com/privacy
  • RevenueCat — gestão de compras — revenuecat.com/privacy
  • Google Firebase — notificações push — firebase.google.com/support/privacy

4. Retenção de dados

Os dados de conta são mantidos até eliminares a conta.

Os dados de recibos são mantidos 90 dias após o fecho de uma sessão.

5. Os teus direitos (RGPD)

Tens direito a aceder, corrigir e eliminar os teus dados.

Podes eliminar a tua conta e todos os dados associados diretamente na app: Perfil → Eliminar Conta.

6. Menores

O serviço não se destina a menores de 13 anos.
